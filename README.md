# Word 格式修改器

本地运行的 Word 文档格式批量修改工具。断网可用，不调用 AI。支持 macOS / Windows。

## 功能

- 上传 `.docx` / `.txt` / `.md` 文件，自动识别标题级别并重新编号
- 套用格式规范（字体、字号、行距、页边距、页码等）
- 字体名称自动从本机字体文件读取，准确匹配 Word 字体名
- 支持多套预设，参数可在界面内调整并保存
- 原文件不动，输出 `原文件名Format.docx`

## 依赖安装

**macOS：**
```bash
pip3 install -r requirements.txt
```
或双击 `start.command`，缺失依赖会自动安装。

**Windows：**
双击 `install.bat`，自动安装所需 Python 包（需已安装 Python 3.9+）。

## 启动方式

### macOS

**方式一（推荐）：双击 `start.command`**
- 双击后自动启动服务，浏览器自动打开 `http://localhost:8080`
- 关闭终端窗口即停止服务

**方式二：命令行启动**
```bash
python3 server.py
```

### Windows

**方式一（推荐）：双击 `start.bat`**
- 双击后自动启动服务，浏览器自动打开 `http://localhost:8080`
- 关闭命令窗口即停止服务

**方式二：命令行启动**
```cmd
python server.py
```

## 使用流程

1. 浏览器打开后，选择格式预设（默认：党政公文标准）
2. 可微调参数，或点"保存当前预设"持久化
3. 拖入 `.docx` / `.txt` / `.md` 文件（支持多选）
4. 点"开始扫描"，在预览页确认标题识别结果
5. 点"确认套用"，等待处理完成
6. 下载格式化后的文件

> 预览确认步骤可勾选"跳过"，以后全自动处理。

## 预设管理

界面内支持：
- 切换预设
- 调整参数后"保存当前预设"
- "另存为新预设"创建自定义规范

预设配置存在 `config.json`，可直接编辑。

## 格式规范说明

当前内置两套预设：

**党政公文标准**
- 文章标题：方正小标宋简体 22pt 加粗居中
- 一级标题：黑体 17pt 左对齐
- 二级标题：楷体 17pt 左对齐
- 三级标题：仿宋 17pt 加粗左对齐
- 正文：仿宋 17pt，首行缩进 2 字符，1.34 倍行距
- 页边距：上 3.8 / 下 3.2 / 左 2.7 / 右 2.7 cm
- 编号：一、（一）、1.

**通用学术格式**
- 文章标题：黑体 16pt 加粗居中
- 正文：仿宋 12pt，首行缩进 2 字符，1.5 倍行距

## 注意事项

- 中文字体依赖本机已安装的字体文件（方正小标宋简体、黑体、楷体、仿宋等）
- 若本机无对应字体，Word 会显示默认字体替代；可在界面"字体设置"中修改字体映射
- 页码为 Word 域，打开文件后按 `F9` 或 `Ctrl+A` → `F9` 更新域
- 暂不支持 `.doc` 格式，请在 Word 中另存为 `.docx` 后使用
